# You Lu Academic Homepage

This directory contains the source for You Lu's academic homepage, adapted from the Minimal Light Jekyll theme.

## Main Files

- `_config.yml`: site metadata, profile links, avatar, and theme settings.
- `index.md`: homepage biography, research interests, publications, and awards includes.
- `_data/publications.yml`: publication list rendered on the homepage.
- `_includes/publications.md`: publication rendering template.
- `_includes/awards.md`: awards section.
- `assets/img/ly.jpg`: profile photo copied from `Leason-ly.github.io/images/ly.jpg`.

## Local Preview

Install the Ruby dependencies and run Jekyll:

```bash
bundle install
bundle exec jekyll serve
```

Then open `http://localhost:4000`.
